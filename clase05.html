Unidad 3: Programación Frontend con Javascript

Curso básico para principiantes

Introducción

En esta unidad aprenderemos los fundamentos de JavaScript, el lenguaje de programación que permite crear interactividad en las páginas web.

Declaración de variables

En JavaScript, podemos declarar variables utilizando las palabras clave var, let y const. Aquí hay un ejemplo:

var nombre = "Juan";
let edad=30; 
const PI = 3.14;

La diferencia entre ellas es que var tiene un alcance global o de función, let tiene un alcance de bloque y const se utiliza para declarar constantes.

Condicionales

Los condicionales nos permiten ejecutar diferentes bloques de código según ciertas condiciones. Aquí hay un ejemplo:

if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {        
    console.log("Eres menor de edad");
}

En este caso, si la variable edad es mayor o igual a 18, se mostrará un mensaje indicando que la persona es mayor de edad; de lo contrario, se mostrará que es menor de edad.

Eventos

Los eventos son acciones que ocurren en la página web, como hacer clic en un botón o mover el mouse. Podemos manejar estos eventos utilizando JavaScript. Aquí hay un ejemplo:

document.getElementById("miBoton").addEventListener("click", function() {    
    alert("¡Botón clickeado!");
});

En este caso, cuando el usuario hace clic en el botón con el ID miBoton, se mostrará una alerta.
